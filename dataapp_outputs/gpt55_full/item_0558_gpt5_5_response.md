We need the linear function through the points \((250,1600)\) and \((650,4200)\).

Slope:
\[
m=\frac{4200-1600}{650-250}=\frac{2600}{400}=\frac{13}{2}
\]

Use \(f(x)=mx+b\) with \((250,1600)\):
\[
1600=\frac{13}{2}(250)+b=1625+b
\]
\[
b=-25
\]

So the function is:
\[
\boxed{f(x)=\frac{13}{2}x-25}
\]
