We need to evaluate the piecewise function at four inputs.

The rule is:
- If \(x>7\), then \(f(x)=\sqrt{x+7}\).
- If \(x\le 7\), then \(f(x)=2\).

Evaluate each:

\(12>7\), so  
\(f(12)=\sqrt{12+7}=\sqrt{19}\).

\(-7\le 7\), so  
\(f(-7)=2\).

\(\pi\le 7\), so  
\(f(\pi)=2\).

\(-7.1\le 7\), so  
\(f(-7.1)=2\).

There are exactly 4 answers, in the requested order: \(f(12), f(-7), f(\pi), f(-7.1)\).

\(\boxed{\sqrt{19}, 2, 2, 2}\)
