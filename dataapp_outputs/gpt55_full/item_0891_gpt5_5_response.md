We need the maximum interval containing \(0\) on which the solution exists.

Separate variables:

\[
\frac{dy}{1+y^2}=dx
\]

Integrate:

\[
\arctan y=x+C
\]

Using \(y(0)=0\):

\[
0=0+C \implies C=0
\]

So

\[
y=\tan x
\]

The function \(\tan x\) is defined on the maximal interval containing \(0\):

\[
\left(-\frac{\pi}{2},\frac{\pi}{2}\right)
\]

This corresponds to option A.

\[
\boxed{A}
